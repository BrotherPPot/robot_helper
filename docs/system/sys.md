# 系统指令

## 登录
指令：`登录`    
示例：
```
登录
```
权限：无  
说明：登录进入Web模式  
***
## 帮助
指令：`帮助|help`    
示例：  
```
帮助   
help  
```    
权限：无  
说明：查看机器人使用帮助
***
